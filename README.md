# Computer Builder

## ✍️ 概要
CPU, GPU, メモリ, ストレージの各パーツを選択し、「仮想のコンピュータ」を組み立て、ゲーミングPCと作業用PCのスコアを算出して比較する比較するアプリケーションです。

## 📎 URL
[Computer Builder](https://computer-builder.mikufolio.site/)

## ⚙️ 使用技術
- **HTML**
- **CSS** 
- **Vanilla JavaScript**
- **AWS**
   - VPC
   - EC2 (Ubuntu 24.04)
